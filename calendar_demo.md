# P.I.L.O.T. 番茄钟日历功能演示

## 🎯 完整工作流程

### 1. 启动自然语言交互模式
```bash
python pilot.py chat -i
```

### 2. 输入工作安排
```
今天的工作任务是：A.完成个人助手agent的开发，B.完成ANR优化验证，C.跟进AI writing需求。重点完成A、B两个任务
```

### 3. 系统自动生成计划
- ✅ 解析任务内容
- ✅ 生成番茄钟时间表
- ✅ 避开午休时间（12:00-14:00）
- ✅ 下午14:10恢复工作

### 4. 选择日历导入方式
```
📅 是否需要创建番茄钟日历？
1. 📱 导入到 iOS/Mac 日历 (自动打开日历应用)
2. 🌐 导入到 Google Calendar (在线同步)
3. 📄 生成 ICS 文件到本地 (手动导入)
4. ❌ 不创建日历

请输入选择 (1-4): 
```

## 🔔 提醒功能详情

### 番茄钟提醒
- **开始前5分钟**: "🍅 番茄钟 #1 即将开始（5分钟后）"
- **开始前1分钟**: "🍅 番茄钟 #1 即将开始（1分钟后）"

### 休息提醒
- **结束前1分钟**: "☕ 番茄休息 即将结束，准备继续工作"

### 任务提醒
- **开始前10分钟**: "📋 完成个人助手agent的开发 即将开始（10分钟后）"

### 午休提醒
- **12:00-14:00**: "午餐 + 休息，14:10 恢复工作"

## 🎨 日历颜色编码

### Google Calendar
- 🔴 **番茄钟** (工作/学习): 红色/蓝色
- 🟢 **休息时间**: 绿色
- 🟡 **具体任务**: 黄色

### ICS文件
- 支持导入到任何兼容的日历应用
- 自动设置提醒
- 包含详细描述和分类

## 📱 使用场景

### iOS/Mac 用户 (推荐)
1. 选择选项1 - **自动体验**
   - ✅ 系统自动打开日历应用
   - ✅ 一键确认导入
   - ✅ 立即获得所有提醒
2. 或选择选项3 - **手动导入**
   - 📄 生成ICS文件
   - 双击文件导入

### Google Calendar 用户
1. 选择选项2
2. 系统自动创建事件
3. 手机/电脑同步显示
4. 获得弹窗提醒

### 其他日历应用
1. 选择选项3
2. 导入生成的 .ics 文件
3. 支持 Outlook, Thunderbird 等
